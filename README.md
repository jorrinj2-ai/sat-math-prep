# SAT Math Prep Platform

A complete SAT Math practice exam platform with student accounts, admin management, SAT-scaled scoring, and detailed analytics.

## Features

- **Student Accounts** — Email/password registration and login
- **80-Question Bank** — Across 4 SAT categories and 3 difficulty levels
- **SAT-Accurate Scoring** — Uses College Board concordance table (200–800 scale)
- **Admin Dashboard** — Create exams, view all students, detailed analytics
- **Progress Tracking** — Score trends, category breakdowns, improvement metrics
- **Assigned Exams** — Admins create exams that appear for all students

## Tech Stack

- **Frontend:** Next.js + React
- **Backend:** Supabase (PostgreSQL + Auth)
- **Hosting:** Vercel (free tier)
